# poke-search

This template should help get you started developing with Vue 3 in Vite.

## Creating the project

```sh
npm init vue@latest
```

### Run the project follow the steps

```sh
cd <name-of-project>
npm install
npm run dev
```

### Install packages

```sh
npm install
```
### run project

```sh
npm run dev
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
